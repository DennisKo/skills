## Available Skills

### grug-brained-reviewer

Find architecture improvements that reduce complexity without adding ceremonial architecture. The skill maps a repository, delegates focused explorer agents when available, connects evidence to grug-brained engineering patterns, and produces a self-contained HTML report of actionable refactoring candidates.

**Use when:**

- Inspecting a project for architecture improvements
- Looking for refactoring candidates that reduce complexity
- Producing an HTML architecture review report
- Applying grug-brained engineering patterns to a codebase
- Preparing to grill and harden a chosen refactoring candidate before implementation

## Install

```sh
npx skills add DennisKo/skills
```

To install only this skill:

```sh
npx skills add DennisKo/skills --skill grug-brained-reviewer
```
